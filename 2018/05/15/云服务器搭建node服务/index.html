配置环境

1.首先安装nodejs(这里以CentOS为例)：参考官网

curl --silent --location https://rpm.nodesource.com/setup_10.x | sudo bash -

sudo yum install -y nodejs

安装yarn（可选）

curl -sL https://dl.yarnpkg.com/rpm/yarn.repo | sudo tee /etc/yum.repos.d/yarn.repo

sudo yum install yarn

2.接下来安装git：参考文章

这里使用yum来安装，这是比较简单快速的方式，缺点是可能安装的版本并不是最新的。如果想要安装最新的版本，可以从源地址下载安装包安装。

sudo yum install git
git --version

3.安装MongoDB

配置yum

vi /etc/yum.repos.d/mongodb-org-3.6.repo

拷贝以下配置到mongodb-org-3.6.repo文件

[mongodb-org-3.6]
name=MongoDB Repository
baseurl=https://repo.mongodb.org/yum/redhat/$releasever/mongodb-org/3.6/x86_64/
gpgcheck=1
enabled=1
gpgkey=https://www.mongodb.org/static/pgp/server-3.6.asc

安装MongoDB

sudo yum install -y mongodb-org

启动、关闭、重启MongoDB

sudo service mongod start | stop | restart

4.安装pm2

npm install pm2 -g

使用配置文件 process.yml

apps:
    - script   : app.js
        instances: 4
        exec_mode: cluster
    - script : worker.js
        watch  : true
        env    :
            NODE_ENV: development
        env_production:
            NODE_ENV: production

启动

pm2 start process.yml

pm2查看，开启，关闭，重启，删除进程

pm2 list

pm2 start app.js

pm2 stop 0
pm2 stop all

pm2 restart 0
pm2 restart all

pm2 delete 0
pm2 delete all

5.使用ngnix映射端口

安装nginx

sudo vi /etc/yum.repos.d/nginx.repo

添加如下配置到nginx.repo

[nginx]
name=nginx repo
baseurl=https://nginx.org/packages/mainline/centos/7/$basearch/
gpgcheck=0
enabled=1

安装nginx

sudo yum update
yum install nginx -y

启动nginx

sudo nginx

配置nginx

添加nginx的server配置文件minimall.conf（nginx目录可能为/usr/local/nginx/conf,/etc/nginx, or /usr/local/etc/nginx）

cd /etc/nginx
mkdir servers
cd servers
vi minimall.conf

添加如下配置

server{
  listen 80;
  server_name 127.0.0.1;
  location / {
    proxy_pass http://127.0.0.1:3000;
  }
}

添加如下配置到nginx配置文件nginx.conf

include /etc/nginx/servers/miniMall.conf;

重启nginx

nginx -s reload
